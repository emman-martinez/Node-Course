# NOC Project 

Objective: To create multiple tasks using a clean architecture with TypeScript

# DEV
1. Copy the .env.template file to .env.
2. Configure the environment variables.
```
PORT=3000
MAILER_EMAIL=
MAILER_SECRET_KEY=
PROD=false
```
3. Execute ```npm install```
4. Run data bases with the command:
```
docker compose up -d
```
5. Execute ```npm run dev```
